💬 قل هذا قبل ما تبدأ

«Trinket هو المكان اللي ستكتبون فيه Python خلال الإجازة. ما يحتاج تثبيت — افتح المتصفح وابدأ. سأكتب أمامكم التمارين الأربعة، وانتم تراقبون وتسألون.»

«لا تقلقوا إذا ما فهمتم كل شيء الآن — هذا طبيعي. المهم تحاولوا وترفعوا ما عملتموه.»

تمرين ١ — المتغيرات والطباعة ٥ دقائق
💬 قل هذا

«المتغير هو صندوق لحفظ قيمة. هنا عندنا صندوق اسمه temp يحفظ درجة الحرارة.»

# تمرين ١: متغيرات وطباعة temp = 34.5 # درجة الحرارة humid = 65 # الرطوبة print("درجة الحرارة:", temp, "°C") print("الرطوبة:", humid, "%") print(f"التقرير: الحرارة {temp}°C والرطوبة {humid}%")
⚙️ بعد الكتابة

اضغط Run ▶ وأرِهم الناتج. «غيّروا الأرقام — ماذا يحدث؟»

تمرين ٢ — الشروط if/elif/else ٧ دقائق
💬 قل هذا

«if تعني "إذا" — البرنامج يقرر ماذا يفعل بناءً على القيمة. هذا بالضبط ما رسمناه في المخطط الانسيابي!»

# تمرين ٢: الشروط temp = 38 if temp > 35: print("🔴 تحذير: حرارة عالية جداً!") elif temp < 10: print("🔵 تحذير: برودة شديدة!") else: print("✅ الحرارة طبيعية")
⚠️ انتبه هنا

«المسافة قبل print مهمة جداً — Python تغضب إذا ما كانت موجودة! هذا يسمى indentation — ٤ مسافات.» — أرِهم الخطأ عمداً ثم صحّحه.

تمرين ٣ — الإدخال input() ٥ دقائق
💬 قل هذا

«هذا التمرين يجعل البرنامج يسألكم — مثل Scratch كان يسأل المستخدم.»

# تمرين ٣: الإدخال من المستخدم temp = float(input("أدخل درجة الحرارة: ")) if temp > 35: print("🔴 حرارة عالية!") else: print("✅ طبيعي")
⚙️ بعد الكتابة

اطلب من طالب أن يقترح رقماً — أدخله وأرِهم النتيجة. «جربوا رقماً كبيراً ورقماً صغيراً.»

تمرين ٤ — الحلقة for ٥ دقائق
💬 قل هذا

«for تعني: كرّر هذا الكود عدة مرات تلقائياً. رصد (Rasd) يستخدمها لقراءة الحساس كل ٣ ثوانٍ!»

# تمرين ٤: الحلقة for for i in range(5): print(f"قراءة رقم {i+1}: البرنامج يعمل!") print("اكتملت القراءات ✓")
🛡️ حارس الغرفة النصي — المشروع المصغّر ٨ دقائق
💬 قل هذا

«هذا البرنامج هو نتيجة التمارين الأربعة مجتمعةً. انظروا — input()، if/else، وطباعة النتائج. هذا رصد (Rasd) بشكل نصي بسيط!»

«في الأسبوع الثاني — ستكتبون هذا بأنفسكم في Trinket. الآن أنا أريكم فقط كيف يبدو النهائي.»

# حارس الغرفة النصي 🛡️ # مشروعك الأول بـ Python الحقيقية temp = float(input("أدخل درجة الحرارة (°C): ")) humid = float(input("أدخل نسبة الرطوبة (%): ")) print("--- تقرير الغرفة ---") if temp > 35: print("🔴 تنبيه: حرارة عالية جداً!") elif temp < 10: print("🔵 تنبيه: برودة شديدة!") else: print("✅ الحرارة طبيعية") if humid > 80: print("💧 تنبيه: رطوبة مرتفعة جداً!") elif humid < 20: print("🏜️ تنبيه: الهواء جاف جداً!") else: print("✅ الرطوبة طبيعية") print("--- انتهى التقرير ---")
