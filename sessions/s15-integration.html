كود ngrok
تشغيل ngrok 🖥
في Terminal — بعد تسجيل الدخول بـ authtoken
# أولاً: أضف الـ authtoken (مرة واحدة فقط)
ngrok config add-authtoken YOUR_TOKEN

# ثانياً: شغّل الـ tunnel
ngrok http 5000

# ستظهر هذه الرسالة:
Forwarding https://abc123.ngrok.io -> localhost:5000

# هذا الرابط يعمل من أي جهاز في العالم!
