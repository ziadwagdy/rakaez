ما الفرق بين if وwhile؟
if: ينفذ مرة واحدة إذا تحقق الشرط — while: يكرر طالما الشرط صحيح
٢
ما الفرق بين for وwhile؟
for: عدد محدد من التكرارات — while: يكرر حتى يتغير الشرط

انظر الفرق — أيهما أسهل في القراءة والتعديل؟
❌ بدون دوال (مكرر) c1 = 25 f1 = c1 * 9/5 + 32 print(f1) c2 = 30 f2 = c2 * 9/5 + 32 print(f2) c3 = 15 f3 = c3 * 9/5 + 32 print(f3)
✅ مع دوال (نظيف) def to_f(c): return c * 9/5 + 32 print(to_f(25)) print(to_f(30)) print(to_f(15))

البنية الأساسية لكل دالة في Python
# تعريف دالة تحويل الحرارة def celsius_to_fahrenheit(c): result = c * 9/5 + 32 return result # استخدام الدالة temp_f = celsius_to_fahrenheit(25) print("25°C =", temp_f, "°F") print(celsius_to_fahrenheit(0)) # 32.0 print(celsius_to_fahrenheit(100)) # 212.0
🔑 def = تعريف الدالة | اسم + (معاملات) | return = النتيجة المُرجعة
دالة متقدمة
دالة التحقق من الحرارة 🌡️
دالة ترجع نصاً مختلفاً حسب درجة الحرارة — مثل رصد (Rasd) تماماً!
def check_temperature(t): """تفحص درجة الحرارة وترجع حالة الغرفة""" if t > 40: return "خطر شديد 🔴" elif t > 30: return "تحذير — مرتفع 🟡" elif t > 15: return "آمن وطبيعي 🟢" else: return "بارد جداً 🔵" print(check_temperature(45)) # خطر شديد 🔴 print(check_temperature(22)) # آمن وطبيعي 🟢
مشروع
محوّل الحرارة الكامل 🌡️↔️
نبني برنامجاً بثلاث دوال — كل دالة مسؤولة عن تحويل واحد
def c_to_f(c): return c * 9/5 + 32 def c_to_k(c): return c + 273.15 def f_to_c(f): return (f - 32) * 5/9 # البرنامج الرئيسي c = float(input("درجة مئوية: ")) print(f"فهرنهايت: {c_to_f(c):.1f}°F") print(f"كلفن: {c_to_k(c):.1f} K")
