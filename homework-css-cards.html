    صمّم الصفحة في style.css

    انسخ الكود التالي كنقطة بداية، ثم غيّر الألوان كما تريد:

    style.css 
    /* ---- خلفية الصفحة كلها ---- */
body {
  background-color: #1a1a2e;
}

/* ---- البطاقة الرئيسية ---- */
.profile-card {
  background-color: #16213e;
  color: #ffffff;
}

/* ---- البطاقات الصغيرة ---- */
.info-card {
  background-color: #0f3460;
  color: #e6edf3;
}

/* ---- حجم الخط ---- */
h1 { font-size: 32px; }
h2 { font-size: 20px; }
h3 { font-size: 16px; color: #8b949e; }
p  { font-size: 14px; }
    🎨 اختر ألوانك: ابحث في Google عن "color picker hex" لتختار اللون الذي تريده وتنسخ الكود (#xxxxxx).

    أضف الألوان في style.css

    الكلاس .card يضبط التصميم المشترك لكل البطاقات، والكلاسات الخاصة تضبط لون كل واحدة:

    style.css — ألوان البطاقات 
    /* ---- التصميم المشترك لكل البطاقات ---- */
.card {
  color: #ffffff;
  font-size: 16px;
}

/* ---- لون خاص لكل حساس ---- */
.card-temp  { background-color: #e03131; } /* أحمر  */
.card-humid { background-color: #1971c2; } /* أزرق  */
.card-rain  { background-color: #0ca678; } /* أخضر  */
.card-soil  { background-color: #2f9e44; } /* أخضر غامق */
    🖌 غيّر الألوان كما تحب! جرّب ألواناً مختلفة لكل حساس.
🏋️ تحدي إضافي

أضف لون مختلف لنص الأرقام داخل كل بطاقة. مثلاً، اجعل رقم درجة الحرارة أصفر اللون. تلميح: أضف داخل .card-temp خاصية color لعنصر h3 باستخدام CSS Selector داخلي:

تلميح
.card-temp h3 {
  color: #ffd43b;
}
💡

مفهوم جديد — font-size حجم الخط

تعلّمه الآن، ستحتاجه في كل مشروع

خاصية font-size تتحكم في حجم النص. الوحدة المستخدمة هي px (بيكسل). كلما كان الرقم أكبر، كان النص أكبر.

font-size: 36px عنوان
font-size: 28px عنوان فرعي
font-size: 20px عنوان صغير
font-size: 14px نص عادي — مناسب للفقرات
font-size: 11px نص صغير — للتفاصيل والملاحظات
مثال — كيف تستخدمها في CSS
h1 { font-size: 36px; }  /* عنوان الصفحة */
h2 { font-size: 24px; }  /* عنوان القسم */
h3 { font-size: 18px; }
p  { font-size: 14px; }  /* نص الفقرات */
🏋️ جرّب الآن

في كلا المشروعين (my-profile و rasd)، عدّل أحجام الخطوط لتبدو الصفحة أجمل. اجعل h1 كبيراً وجذاباً، وp صغيرة ومريحة للقراءة.
