أول Flask App
Hello Flask! 👋
أبسط سيرفر ممكن — ٦ أسطر فقط
# pip install flask
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/')
def home():
  return 'مرحباً من رصد (Rasd)!'

@app.route('/api/status')
def status():
  return jsonify({"status": "online", "guardian": True})

if __name__ == '__main__':
  app.run(debug=True)

نشاط — تعديل
أضف Route جديداً ✏️
في app.py — أضف هذا الـ endpoint ثم اختبره في المتصفح
@app.route('/api/hello')
def hello():
  return jsonify({
    "message": "أهلاً من رصد (Rasd)!",
    "version": "1.0",
    "guardian": "active"
  })

# اختبر: http://localhost:5000/api/hello
🎯 هدف إضافي: أضف اسمك في الـ JSON — اجعله يقول "مرحباً من [اسمك]"
