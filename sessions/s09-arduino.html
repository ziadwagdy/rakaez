المنافذ الرقمية — Digital Pins (0-13)
تُرسل أو تستقبل إشارات رقمية (0 أو 1 = 0V أو 5V) — Pin 13 فيه LED مدمج!

تقرأ قيماً بين 0-1023 من الحساسات التناظرية مثل حساس الضوء والحرارة
⚡

5V: جهد خرج للمكونات | GND: الأرضي | Vin: مدخل طاقة خارجية
USB
منفذ USB + زر Reset
USB: لرفع الكود وتغذية الطاقة | Reset: يُعيد تشغيل البرنامج من البداية
أول برنامج
Blink — الـ LED يومض! 💡
تقليد Arduino — أول كود تكتبه هو Blink — مثل "Hello World" تماماً!
// Blink — أول برنامج Arduino // كل برنامج Arduino يحتوي على دالتين void setup() { // تعمل مرة واحدة عند بدء التشغيل pinMode(13, OUTPUT); // Pin 13 = خرج } void loop() { // تتكرر إلى الأبد digitalWrite(13, HIGH); // ضوء يشتعل delay(1000); // انتظر ثانية digitalWrite(13, LOW); // ضوء ينطفئ delay(1000); // انتظر ثانية }
▶️ ارفع الكود إلى Arduino وشاهد الـ LED المدمج يبدأ بالوميض كل ثانية!
تطبيق
عدّل توقيت الوميض ⏱️
بتغيير رقم واحد — تتغير سرعة الوميض كلياً!
١
وميض سريع — 200ms
غيّر delay(1000) إلى delay(200) — شاهد كيف يومض بسرعة شبه مستمرة
٢
وميض بطيء — 2000ms
غيّر delay إلى delay(2000) — الآن كل وميض يستغرق ٢ ثانية كاملة
٣
وميض غير متساوٍ — SOS!
HIGH 200ms → LOW 200ms × 3 | HIGH 600ms × 3 | HIGH 200ms × 3 — نمط SOS الاستغاثة!
💡 تحدي: هل يمكنك برمجة إشارة مرور؟ أخضر 5 ثانية → أصفر 2 → أحمر 5!
المدخلات
قراءة الزر — Digital Input 🔘
Arduino لا يُرسل فقط — يستقبل أيضاً! نقرأ حالة زر ضغط
// LED يضيء فقط عند الضغط على الزر void setup() { pinMode(13, OUTPUT); // LED pinMode(2, INPUT); // الزر على Pin 2 } void loop() { int btn = digitalRead(2); // اقرأ حالة الزر if (btn == HIGH) { digitalWrite(13, HIGH); // ضغطت: أضيء } else { digitalWrite(13, LOW); // لم تضغط: أطفئ } }
🤯 لاحظوا: نفس مفهوم if/else الذي تعلمناه في Python — الكود يبدو مختلفاً لكن المنطق واحد!
التواصل
كيف يتحدث Arduino مع Python؟ 🔗
Serial Communication — Arduino يُرسل البيانات عبر USB لـ Python
// Arduino — يقرأ الحرارة ويُرسل لـ Python void setup() { Serial.begin(9600); // ابدأ التواصل } void loop() { float temp = readSensor(); Serial.println(temp); // أرسل الرقم delay(60000); // كل دقيقة } // Python يستقبل: // import serial // data = serial_port.readline() // temperature = float(data)
