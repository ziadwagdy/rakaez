الشروط
جملة if / elif / else 🔀
نجعل البرنامج يقرر ماذا يفعل بناءً على الظروف
# مثال: تقييم درجة الحرارة temp = int(input("أدخل درجة الحرارة: ")) if temp > 35: print("خطر! حار جداً 🔴") elif temp > 25: print("دافئ ومريح 🟡") elif temp > 15: print("معتدل وجميل 🟢") else: print("بارد! ارتدِ معطفاً 🔵")
⚠️ مهم: المسافة البادئة (Indentation) إلزامية في Python — 4 مسافات بعد كل if/elif/else

if/else في رصد (Rasd) 🛡️
هذا بالضبط كيف يقرر النظام متى يرسل تنبيهاً!
# كود حقيقي من رصد (Rasd) temperature = read_sensor() # يقرأ من الحساس humidity = read_humidity() # يقرأ الرطوبة if temperature > 40: send_alert("🔴 درجة حرارة خطيرة!") elif temperature > 30: send_alert("🟡 درجة حرارة مرتفعة") elif humidity > 80: send_alert("💧 رطوبة عالية جداً!") else: log("✅ كل شيء طبيعي")
🤯 كل تنبيه تراه في تطبيق رصد (Rasd) صدر من كود if/else مثل هذا!
الحلقات — النوع الأول
حلقة for 🔄
تكرر الكود لعدد محدد مسبقاً من المرات
# اطبع الأرقام من 0 إلى 9 for i in range(10): print("الرقم:", i) # اطبع الأرقام الزوجية فقط for i in range(0, 20, 2): print(i, "زوجي ✓") # قرأة قراءات حساس 5 مرات for reading in range(5): temp = read_sensor() print(f"القراءة {reading+1}: {temp}°C")
📝 range(start, stop, step): البداية، النهاية، الخطوة — كلها اختيارية عدا النهاية
الحلقات — النوع الثاني
حلقة while ⏳
تكرر الكود طالما الشرط صحيح — مثالية للألعاب!
# المراقبة المستمرة في رصد (Rasd) system_on = True while system_on: temp = read_sensor() if temp > 40: send_alert("خطر!") if temp == -999: # كود الإيقاف system_on = False sleep(60) # انتظر دقيقة
⚠️ تحذير: تأكد دائماً أن الشرط سيصبح False في نهاية ما — وإلا سيعمل البرنامج إلى الأبد!
مشروع اليوم
لعبة تخمين الرقم 🎯
نبني لعبة كاملة تجمع: متغيرات + شروط + حلقة while
import random secret = random.randint(1, 100) print("خمّن رقماً بين 1 و 100!") guess = 0 while guess != secret: guess = int(input("تخمينك: ")) if guess < secret: print("⬆️ أعلى من ذلك!") elif guess > secret: print("⬇️ أقل من ذلك!") print("🎉 صحيح! أحسنت!")
