# الجلسة ٦ — الإلكترونيات والدوائر

## المفاهيم الأساسية

| المصطلح | المعنى |
|---------|--------|
| Voltage (الفولت) | ضغط الكهرباء — مثل ضغط الماء في الأنبوب |
| Current (التيار) | كمية الكهرباء المتدفقة |
| Resistance (المقاومة) | ما يُقاوم تدفق الكهرباء |
| GND | الأرضي — نقطة الإغلاق |
| 3.3V | مصدر الطاقة في Pico W |

## مكونات اليوم
- Raspberry Pi Pico W
- LED (ثنائي باعث للضوء)
- مقاومة 330Ω
- لوحة التجارب (Breadboard)
- أسلاك توصيل

## دائرة LED البسيطة
```
Pico W GP15 → مقاومة 330Ω → LED (الساق الطويلة) → GND
```

## قاعدة أوم
```
V = I × R
الفولت = التيار × المقاومة
```
