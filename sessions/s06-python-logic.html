الحلقات — النوع الأول
حلقة for 🔄
تكرر الكود لعدد محدد مسبقاً من المرات — مثالية لقراءات الحساسات!
# اطبع الأرقام من 1 إلى 5 for i in range(1, 6): print(f"العدد: {i}") # محاكاة 5 قراءات من الحساس import random for i in range(5): temp = round(random.uniform(20, 40), 1) print(f"القراءة {i+1}: {temp}°C") # range(start, stop, step) — الخطوة اختيارية for t in range(0, 100, 10): # 0,10,20...90 print(f"درجة: {t}")
📝 range(stop): من 0 إلى stop-1 | range(start, stop): من start إلى stop-1

بناء الدوال
def — تعريف دالة 🔧
هيكل الدالة دائماً نفسه — اسم، مدخلات، كود، مخرج (return)
# دالة بسيطة def greet(name): print(f"أهلاً يا {name}! 👋") greet("زياد") # → أهلاً يا زياد! greet("سارة") # → أهلاً يا سارة! # دالة مع return — تُرجع قيمة def check_sensor(value, min_val, max_val): if value > max_val: return "HIGH" elif value < min_val: return "LOW" else: return "OK" # الاستخدام status = check_sensor(38, 10, 35) print(f"حالة الحرارة: {status}") # → HIGH
💡 check_sensor() هذه هي نفس الدالة الموجودة في alerts.py في رصد (Rasd)!
