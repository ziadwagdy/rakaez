نفس HTML — فقط أضفنا CSS

CSS = Cascading Style Sheets — ملف منفصل يتحكم في شكل كل عنصر HTML
الأساسيات
كيف نكتب CSS
/* نختار العنصر ثم نضع الخصائص بين {} */

h1 {
  color: #00e5c0;        /* لون النص */
  font-size: 32px;       /* حجم الخط */
  text-align: center;    /* توسيط */
}

.card {                  /* . تعني class */
  background: #161b22;
  border-radius: 12px;
  padding: 24px;
}

#temp {                  /* # تعني id */
  font-size: 48px;
  font-weight: 900;
}

اسم العنصر

h1 p div — يطبق على كل نسخ العنصر

.class

يطبق على كل عنصر بهذا الـ class — الأكثر استخداماً

#id

يطبق على عنصر واحد فقط — اسم فريد في الصفحة

مفهوم أساسي
Box Model — كل عنصر صندوق 📦

الأربعة طبقات

margin — المسافة الخارجية
border — الحدود
padding — المسافة الداخلية
المحتوى

في الكود

.card {
  /* المحتوى */
  width: 200px;

  /* padding — داخل */
  padding: 24px;

  /* border — الحدود */
  border: 1px solid #30363d;
  border-radius: 12px;

  /* margin — خارج */
  margin: 16px;
}
التصميم
الألوان في CSS 🎨
/* ثلاث طرق لكتابة الألوان */

.card {
  /* ١. اسم اللون */
  color: white;

  /* ٢. Hex — الأكثر شيوعاً */
  background: #161b22;
  border-color: #30363d;

  /* ٣. CSS Variables — الأذكى */
  color: var(--accent);
}

/* تعريف المتغيرات مرة واحدة */
:root {
  --bg:     #0d1117;
  --card:   #161b22;
  --accent: #00e5c0;   /* أخضر-فيروزي — رصد */
  --blue:   #58a6ff;
  --amber:  #f0b429;
  --red:    #ff6b6b;
}
CSS Variables — عرّف اللون مرة واحدة في :root واستخدمه في كل مكان. تريد تغيير اللون؟ غيّره مرة واحدة فقط.
التخطيط
CSS Grid — أربع بطاقات في صف

الكود

.dashboard {
  display: grid;

  /* 4 أعمدة بعرض متساوٍ */
  grid-template-columns: 1fr 1fr 1fr 1fr;

  /* مسافة بين البطاقات */
  gap: 16px;

  /* توسيط + حد أقصى للعرض */
  max-width: 900px;
  margin: 0 auto;
  padding: 24px;
}

تطبيق عملي
CSS كامل لبطاقة حساس واحدة
/* ── الصفحة ── */
body {
  background: #0d1117;
  color: #e6edf3;
  font-family: 'Cairo', sans-serif;
  margin: 0;
  padding: 24px;
}

/* ── العنوان ── */
h1 {
  text-align: center;
  color: #00e5c0;
  font-size: 28px;
  margin-bottom: 32px;
}

/* ── شبكة البطاقات ── */
.dashboard {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 16px;
  max-width: 900px;
  margin: 0 auto;
}

/* ── البطاقة ── */
.card {
  background: #161b22;
  border: 1px solid #30363d;
  border-radius: 12px;
  padding: 24px 16px;
  text-align: center;
}

.card .label { font-size: 13px; color: #8b949e; margin-bottom: 12px; }
.card .value { font-size: 44px; font-weight: 900; color: #00e5c0; }
.card .unit  { font-size: 13px; color: #8b949e; margin-top: 6px; }
إضافات
Hover و Transitions — تأثيرات بسيطة
/* تأثير عند تمرير الماوس على البطاقة */
.card {
  background: #161b22;
  border: 1px solid #30363d;
  border-radius: 12px;
  padding: 24px;

  /* الانتقال السلس — 0.2 ثانية */
  transition: border-color 0.2s, transform 0.2s;
}

.card:hover {
  border-color: #00e5c0;          /* الحدود تضيء */
  transform: translateY(-4px);    /* تطفو للأعلى قليلاً */
}

/* لون مختلف لكل بطاقة */
.card-temp  { border-right: 3px solid #ff6b6b; }
.card-humid { border-right: 3px solid #58a6ff; }
.card-water { border-right: 3px solid #3fb950; }
.card-soil  { border-right: 3px solid #f0b429; }
transition يجعل التغيير تدريجياً بدلاً من فوري — ضعه دائماً على العنصر الأصلي وليس على :hover
