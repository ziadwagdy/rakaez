دالة setup()

تعمل مرة واحدة عند التشغيل — نضع فيها إعدادات المنافذ وتهيئة الحساسات والمكتبات

🔁

دالة loop()

تعمل للأبد في حلقة لانهائية — هنا نقرأ الحساسات ونُرسل البيانات ونتحكم في الأجهزة

💡

LED Blink

أول مشروع — كتبنا كوداً يُضيء LED ثم يُطفئها كل ثانية باستخدام digitalWrite و delay

📺

Serial Monitor

نافذة الاتصال بين Arduino والكمبيوتر — Serial.begin(9600) لفتحها و Serial.println() للطباعة

مفهوم جديد
ما هي الحساسات؟
الحساسات هي عيون وآذان الجهاز — تُحوّل الظواهر الفيزيائية إلى إشارات رقمية يفهمها Arduino
🔘

حساس رقمي

يُعطي قيمتين فقط: 0 أو 1 — مثل: مفتاح الضغط، حساس المطر وحساس التربة، حساس الباب

📈

حساس تناظري

يُعطي قيمة مستمرة من 0 إلى 1023 — مثل: حساس الضوء LDR، حساس الصوت، الجهد المتغير

🌡

DHT11

حساس ذكي يقيس درجة الحرارة من ‑40° إلى 80° والرطوبة من 0% إلى 100% بدقة ±0.5%

📳

حساس المطر وحساس التربة

يكتشف الاهتزازات والصدمات — يُرسل HIGH عند الاهتزاز و LOW عند السكون التام

التوصيل والكود
توصيل DHT11 بـ Arduino
DHT11 له 4 أرجل — نوصّل 3 منها ونترك Pin 3 فارغاً
1
Pin 1 → VCC (5V)
طرف الطاقة — وصّله بمنفذ 5V على Arduino لتشغيل الحساس
2
Pin 2 → Digital Pin 2 (DATA)
طرف البيانات — هنا تأتي قراءات الحرارة والرطوبة. ضع مقاومة 10kΩ بينه وبين 5V
4
Pin 4 → GND
طرف الأرضي — يُغلق الدائرة الكهربائية ويُثبّت الجهد
#include <DHT.h> DHT dht(2, DHT11); void setup() { Serial.begin(9600); dht.begin(); } void loop() { float temp = dht.readTemperature(); float hum = dht.readHumidity(); Serial.print("Temp: "); Serial.print(temp); Serial.println(" C"); Serial.print("Hum: "); Serial.print(hum); Serial.println(" %"); delay(2000); }
تجربة عملية
عرض قراءات DHT11 على Serial Monitor
بعد رفع الكود، افتح Serial Monitor من Tools ← Serial Monitor — ستظهر القراءات كل ثانيتين
Temp: 23.50 C Hum: 61.20 % Temp: 23.60 C Hum: 61.00 % Temp: 24.10 C Hum: 60.80 % Temp: 24.30 C Hum: 59.50 %
💡 جرّب: ضع إصبعك على DHT11 — ستلاحظ ارتفاع درجة الحرارة فوراً!
أو انفخ عليه بلطف لترى تغيير قراءة الرطوبة.
الحساس الثاني
حساس المطر وحساس التربة 📳
نُضيف حساس المطر وحساس التربة على المنفذ الرقمي 3 — سيُنبّهنا عند أي حركة أو اهتزاز في الغرفة
// حساس المطر وحساس التربة على Pin 3 void setup() { Serial.begin(9600); pinMode(3, INPUT); } void loop() { int vib = digitalRead(3); if (vib == HIGH) { Serial.println("تحذير: تم رصد اهتزاز!"); } else { Serial.println("الغرفة هادئة..."); } delay(500); }
🔧 توصيل بسيط: VCC → 5V | OUT → Pin 3 | GND → GND
اطرق الطاولة بلطف وانظر ماذا يحدث في Serial Monitor!
