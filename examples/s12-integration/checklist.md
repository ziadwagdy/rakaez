# قائمة التكامل الكامل — رصد

## ترتيب التشغيل
1. [ ] شغّل Flask: `cd room-guardian && python app.py`
2. [ ] افتح http://localhost:8080 — تحقق أن البيانات تتغير
3. [ ] افتح dashboard.html — تحقق أن البطاقات تتحدث
4. [ ] وصّل Pico W — تحقق أن Flask يستقبل POST

## تحقق من كل مكون
- [ ] `/api/latest` يعود ببيانات
- [ ] `/api/readings` يعود بسجل القراءات
- [ ] `/api/alerts` يعود بالتنبيهات
- [ ] الرسم البياني يرسم النقاط
- [ ] التنبيه يظهر عند الحرارة العالية
- [ ] Pico W يرسل كل 5 ثوانٍ

## مشاكل شائعة
| المشكلة | الحل |
|---------|------|
| البطاقات لا تتحدث | تأكد أن Flask يعمل على port 8080 |
| CORS error | تأكد من `from flask_cors import CORS` |
| Pico W لا يتصل | تأكد من IP الحاسوب في main.py |
| الرسم لا يظهر | تأكد من تحميل Chart.js |
