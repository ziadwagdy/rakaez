جميع المهام تُنفَّذ داخل Visual Studio Code

كل ما ستكتبه من HTML وCSS يجب أن يكون داخل VS Code على جهازك. لا يُقبل العمل داخل المتصفح مباشرةً أو في أي محرر آخر. إذا لم يكن VS Code مثبتاً على جهازك، حمّله من code.visualstudio.com وثبّته قبل البدء.

المهمة الأولى — عرض تقني 01

اختيار موضوع تقني وتقديمه أمام الجميع
الهدف: كل مجموعة تختار موضوعاً يتعلق بالتقنية أو إنترنت الأشياء (IoT)، تبحث عنه وتقدّمه أمام الجميع في الجلسة القادمة.
  • 1

    اختاروا الموضوع معاً

    تشاور أفراد المجموعة واختاروا موضوعاً يثير اهتمام الجميع. الموضوع يجب أن يكون له علاقة بالتقنية، الروبوتات، أو إنترنت الأشياء (IoT). مثال على مواضيع ممكنة:

    🏠
    المنازل الذكية
    Smart Home
    🌱
    الزراعة الذكية
    Smart Agriculture
    🚗
    السيارات ذاتية القيادة
    Self-Driving Cars
    🏥
    الصحة الرقمية
    Digital Health / Wearables
    🤖
    الروبوتات الصناعية
    Industrial Robots
    🌍
    المدن الذكية
    Smart Cities
  • 2

    ابحثوا عن المعلومات

    ابحثوا عن الموضوع على الإنترنت وأجيبوا على هذه الأسئلة في عرضكم:

    • ما هو هذا الموضوع وكيف يعمل؟
    • أين يُستخدم في الحياة اليومية؟
    • ما هي فائدته للناس أو للمجتمع؟
    • ما هو مثال حقيقي على منتج أو مشروع في هذا المجال؟
  • 3

    جهّزوا العرض

    يمكنكم استخدام PowerPoint، أو Canva، أو حتى الكتابة على ورقة إذا احتجتم. المهم أن كل فرد في المجموعة يعرف الموضوع ويستطيع الشرح أمام الجميع.

    ⏱ مدة العرض: من 3 إلى 5 دقائق لكل مجموعة. ستعرضون في الجلسة القادمة.

المهمة الثانية — الصفحة الأولى لمشروع رصد 02

تطبيق HTML وCSS على واجهة مشروعك
الهدف: طبّق ما تعلمته (div، العناوين، ملف CSS، الألوان والخلفيات) لبناء الصفحة الرئيسية لمشروع رصد — بدون أي تصميم متقدم، فقط HTML + CSS أساسي.
تذكير بما تعلمناه حتى الآن: عناوين h1–h6، الفقرات p، الـ div، إنشاء ملف CSS منفصل وربطه بـ HTML، خاصية color لتلوين النص، خاصية background-color لتلوين الخلفية.
  • 1

    أنشئ مجلداً جديداً للمشروع

    أنشئ مجلداً جديداً اسمه rasd على سطح المكتب (كليك يمين على سطح المكتب → New Folder على Windows، أو New Folder على Mac). داخله أنشئ ملفين:

    • ملف HTML باسم index.html
    • ملف CSS باسم style.css
    📁 افتح VS Code ثم: File → Open Folder → اختر مجلد rasd من سطح المكتب. بعدها أنشئ الملفين بالضغط على أيقونة الملف الجديد (New File) في الشريط الجانبي على اليسار.
  • 2

    ابنِ الهيكل الأساسي في index.html

    الصفحة تحتاج ثلاثة أقسام رئيسية:

    index.html — الهيكل الأساسي
    <!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
  <meta charset="UTF-8">
  <title>رصد — مراقبة البيئة</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <!-- الرأس -->
  <div class="header">
    <h1>🛡 رصد</h1>
    <p>نظام مراقبة البيئة</p>
  </div>

  <!-- بطاقات الحساسات -->
  <div class="cards">

    <div class="card">
      <h2>🌡️ درجة الحرارة</h2>
      <p>Sensor: Temperature</p>
      <h3>-- °C</h3>
    </div>

    <div class="card">
      <h2>💧 الرطوبة</h2>
      <p>Sensor: Humidity</p>
      <h3>-- %</h3>
    </div>

    <div class="card">
      <h2>🌧️ مستوى المطر</h2>
      <p>Sensor: Rain</p>
      <h3>--</h3>
    </div>

    <div class="card">
      <h2>🌱 رطوبة التربة</h2>
      <p>Sensor: Soil Moisture</p>
      <h3>--</h3>
    </div>

  </div>

</body>
</html>
    لا تنسَ dir="rtl" في عنصر html لأن موقعنا بالعربية.
  • 3

    أضف التصميم في style.css

    الآن طبّق الألوان والخلفيات على الأقسام. يمكنك تغيير الألوان لما يعجبك — هذا مجرد مثال للبدء:

    style.css — التصميم الأساسي
    /* ---- الصفحة كلها ---- */
body {
  background-color: #f0f4f8;
  color: #1a1a2e;
}

/* ---- الرأس ---- */
.header {
  background-color: #1a1a2e;
  color: #ffffff;
}

/* ---- قسم البطاقات ---- */
.cards {
  background-color: #e8f4fd;
}

/* ---- كل بطاقة حساس ---- */
.card {
  background-color: #ffffff;
  color: #0d1117;
}
    🎨 اختر ألوانك الخاصة! يمكنك استخدام الاسم (مثل red, blue) أو كود الـ HEX (مثل #ff6b6b). ابحث في Google عن "color picker" لتختار اللون الذي يعجبك.
  • 4

    افتح الملف في المتصفح وشوف النتيجة

    اضغط كليك يمين على ملف index.html في VS Code → اختر Open with Live Server. إذا لم تجد هذا الخيار، افتح مجلد rasd في File Explorer (على Windows) أو Finder (على Mac)، ثم اضغط مرتين على الملف وسيفتح في المتصفح.

    💡 في كل مرة تحفظ فيها التعديلات (Ctrl+S على Windows · Cmd+S على Mac) ستُحدَّث الصفحة تلقائياً إذا استخدمت Live Server.
  • 5

    التسليم — ارفع المجلد على Drive

    بعد الانتهاء، ارفع مجلد rasd كاملاً (يحتوي على index.html و style.css) داخل مجلدك الخاص في Google Drive.

    📁 افتح مجلد واجبات رصد على Drive
    ابحث عن مجلد اسمه اسمك (مثل "أحمد" أو "سارة") وارفع المجلد بالكامل بداخله.
كيف ستبدو الصفحة (مثال مبسّط):
