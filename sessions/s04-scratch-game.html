المفهوم الأول
الشروط في Scratch — If/Then 🔀
الشرط يجعل اللعبة تتخذ قرارات — هل لمست العدو؟ هل وصلت للهدف؟
// عندما تلمس القطة العقبة الحمراء Forever If <touching [عقبة]?> Then Say "Game Over! 💥" for 2 secs Stop [all]
✅

If / Then

إذا تحقق الشرط، نفّذ الأوامر داخل الكتلة

🔄

If / Else

إذا تحقق: نفّذ هذا — وإلا: نفّذ ذاك

🔁

Forever

كرر الشرط باستمرار — أساس كل لعبة

الحلقات في Scratch 🔁
الحلقة تكرر الأوامر — بدونها ستتوقف اللعبة بعد ثانية!
// حلقة اللعبة الرئيسية — تعمل إلى الأبد Forever Move 3 steps If <on edge?> Then Bounce if on edge If <touching [نجمة]?> Then Change [score] by 1
🔁 Repeat vs Forever: استخدم Repeat عندما تريد عدداً محدداً من التكرارات، وForever للأشياء التي لا تتوقف أبداً مثل حركة اللعبة
إضافة النقاط
متغير Score — النقاط! 🏆
المتغير هو "صندوق" نحفظ فيه رقم النقاط ونغيره أثناء اللعبة
١
أنشئ المتغير
Variables → Make a Variable → اكتب "score" → OK
٢
اضبطه في البداية
When Green Flag Clicked → Set [score] to 0
٣
زِد النقاط عند الاصطدام
If touching [نجمة] → Change [score] by 1 → Play Sound "Ding"
٤
انتهاء اللعبة
If [score] = 10 → Say "فزت! 🎉" → Stop All
