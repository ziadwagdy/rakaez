جداول منظّمة للبيانات

📋 الجدول (Table)

مثل ورقة Excel — صفوف وأعمدة

📄 الصف (Row)

سجل واحد — قراءة مجسات واحدة

🏷️ العمود (Column)

نوع البيانات — temperature · humidity · timestamp

🔑 المفتاح (id)

رقم فريد لكل صف — لا يتكرر أبداً

SQL أساسيات
أربعة أوامر تكفي
-- إنشاء الجدول
CREATE TABLE readings (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    temperature REAL,
    humidity    REAL,
    timestamp   TEXT DEFAULT CURRENT_TIMESTAMP
);

-- إضافة صف
INSERT INTO readings (temperature, humidity) VALUES (28.5, 65.0);

-- قراءة البيانات
SELECT * FROM readings WHERE temperature > 30;

-- آخر ١٠ قراءات
SELECT * FROM readings ORDER BY id DESC LIMIT 10;
SQLite في Python
Flask + SQLite معاً
import sqlite3

# اتصال بقاعدة البيانات
conn = sqlite3.connect('data.db')
cursor = conn.cursor()

# أدخل قراءة جديدة
cursor.execute(
    'INSERT INTO readings (temperature, humidity) VALUES (?, ?)',
    (28.5, 65.0)
)
conn.commit()

# اقرأ آخر قراءة
cursor.execute('SELECT * FROM readings ORDER BY id DESC LIMIT 1')
row = cursor.fetchone()
print(row)
