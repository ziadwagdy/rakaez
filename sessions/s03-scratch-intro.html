الشروط — نفس القرار 🔀
حارس الغرفة: إذا كانت الحرارة عالية — أرسل تحذيراً

🐱 Scratch

ask "الحرارة؟" and wait
if answer > 35 then
   say "🔴 تحذير حار!"
else
   say "✅ طبيعي"

🐍 Python

temp = float(input("الحرارة؟ "))

if temp > 35:
    print("🔴 تحذير حار!")
else:
    print("✅ طبيعي")

🐱 Scratch

set i to 1
repeat 10
   say join "قراءة رقم " i
   change i by 1

🐍 Python

for i in range(1, 11):
    print(f"قراءة رقم {i}")
