    اكتب الـ CSS

    style.css 
    /* الحاوية — هنا تُفعَّل الـ Flex */
.container {
  display: flex;
  justify-content: space-evenly;
}

/* تصميم مشترك لكل الصناديق */
.box {
  color: #ffffff;
  text-align: center;
  font-size: 18px;
  font-weight: 700;
  padding: 30px 24px;
  border-radius: 10px;
  min-width: 140px;
}

/* لون خاص لكل صندوق */
.box-blue  { background-color: #0366d6; }
.box-green { background-color: #0a7c5c; }
.box-amber { background-color: #b45309; }
    🔍 لاحظ أن display: flex و justify-content يُكتبان على الحاوية (.container) وليس على الصناديق نفسها.

    شوف النتيجة وجرّب التعديل

    افتح الصفحة بـ Live Server. بعد ما تشوف الصناديق الثلاثة مصطفّة، جرّب تغيير justify-content من space-evenly إلى center أو space-between وشوف الفرق.

    ابنِ هيكل الصفحة في index.html

    الصفحة مكوّنة من قسمين: الرأس (.header) ومنطقة البطاقات (.cards). بطاقات الحساسات الأربع تذهب بداخل .cards:

  <title>رصد — مراقبة البيئة</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <!-- ===== الرأس ===== -->
  <div class="header">
    <h1>🛡 رصد</h1>
    <p>نظام مراقبة البيئة</p>
  </div>

  <!-- ===== البطاقات ===== -->
  <div class="cards">

    <div class="card card-temp">
      <h2>🌡️</h2>

    صمّم الصفحة في style.css

    هنا يحدث السحر — display: flex على .cards يجعل البطاقات الأربع تصطف في صف واحد:

    style.css — التصميم الكامل 
    /* ===== الصفحة ===== */
body {
  background-color: #f0f4f8;
  margin: 0;
}

/* ===== الرأس ===== */
.header {
  background-color: #1a1a2e;
  text-align: center;
  padding: 24px;
}

.header h1 {
  color: #ffffff;
  font-size: 32px;
  font-weight: 900;
}

.header p {
  color: #8b949e;
  font-size: 14px;
}

/* ===== حاوية البطاقات — هنا Flex ===== */
.cards {
  display: flex;
  justify-content: space-evenly;
  padding: 32px 16px;
}

/* ===== كل بطاقة ===== */
.card {
  background-color: #ffffff;
  text-align: center;
  padding: 24px 20px;
  border-radius: 12px;
  min-width: 150px;
}

.card h2 { font-size: 32px; }           /* الإيموجي */
.card h3 { font-size: 14px; font-weight: 700; }
.card p  { font-size: 11px; color: #6b7fa3; }
.card h1 { font-size: 28px; font-weight: 900; }

/* ===== لون القراءة لكل بطاقة ===== */
.card-temp  h1 { color: #e03131; }
.card-humid h1 { color: #1971c2; }
.card-rain  h1 { color: #0ca678; }
.card-soil  h1 { color: #2f9e44; }
    💡 display: flex على .cards يجعل أبناءها المباشرين (البطاقات الأربع) يصطفون تلقائياً في صف. بدونه ستكون البطاقات فوق بعضها.

    قارن نتيجتك بالهدف أعلاه

    افتح صفحتك بـ Live Server وقارنها بالمثال الأخضر أعلاه. هل البطاقات الأربع في صف واحد؟ هل النص مركّز؟ هل الأرقام ظهرت بألوان مختلفة؟

    تذكير: ما يهم ليس أن تكون الألوان نفسها بالضبط — المهم أن يكون التصميم بنفس الهيكل: رأس في الأعلى + أربع بطاقات في صف واحد.
